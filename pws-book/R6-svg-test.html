R6 — inline vector SVG, proof of concept

Test file — not part of the book. These three figures are inline vector SVG exported straight from PWSEditor via the MCP (pws_export svg), then post-processed only to drop the editor grid and the green anchor-dots. Everything else — states, transitions, labels, dashboards, and the black / red / blue action colouring — is genuine vector from the tool, and each case is verified by pws_get_report.
