PART-WHOLE STATECHARTS: THEORY, DESIGN, PRACTICE
Complete HTML Tutorial
==================================================

ABOUT THIS BOOK
---------------

This is a comprehensive, self-contained tutorial on Part-Whole Statecharts (PWS)
—a formal methodology for designing and verifying hierarchical reactive systems.

The book covers:
  • Motivation and the problems PWS solves (Introduction, Part I)
  • Core formalism and semantics (Parts II–III)
  • Design methodology and verification (Parts IV–V)
  • Hands-on practice with PWSEditor (Part VI)
  • Reflection and future directions (Postface)

STRUCTURE
---------

index.html
  Master index with complete table of contents, overview of the four
  workflow threads (TEORIA, SCRITTURA SCIENTIFICA, PROGRAMMAZIONE TOOL,
  DIDATTICA), and key concepts at a glance.

introduction.html
  Overview of PWS philosophy, why it matters, and how the book is organized.

part-1-the-problem.html
  The motivation: state explosion, mutual control, livelock, deadlock,
  and why traditional verification is late and costly.

part-2-the-pws-model.html
  Hierarchical decomposition: assemblies, components, controllers,
  interfaces, and the part-whole relationship.

part-3-configuration-semantics.html
  Giving meaning to controller states: specified semantics, obtained
  semantics, reactive spaces, and exit zones.

part-4-designing-correct-controllers.html
  The Correct-by-Construction methodology: how to design controllers
  incrementally with correctness emerging from design itself.

part-5-compositional-verification.html
  Hierarchical verification: contracts, composition, lifting, hiding,
  and how local proofs compose into global correctness.

part-6-building-controllers.html
  Practical tutorial: using PWSEditor to build the Cover + Laser
  controller step by step, reading dashboards, and closing exit zones.

postface.html
  Synthesis: unifying principles, relationship to classical theory,
  open problems, and future research directions.

FILONI-REFERENCE.txt
  Reference document describing the four workflow threads and how to
  approach requests in different contexts.

ABOUT THE PROJECT
-----------------

Part-Whole Statecharts is a methodology developed for designing reactive
controllers with formal correctness guarantees. This tutorial covers:

  • Formal foundations in supervisory control theory (RW theory)
  • Hierarchical decomposition and compositional verification
  • Integration with temporal logic (LTL) for property specification
  • Implementation via the PWSEditor tool

For more information, code, papers, and resources:
  GitHub: https://github.com/lucapazzi/PWSEditor-4.0
